🌿 Hidden Gem Bakti – Website Informasi & Sistem Data
📌 Tentang Project

Hidden Gem Bakti adalah sebuah website berbasis PHP Native yang digunakan untuk mengelola dan menampilkan data sistem (seperti peserta, ranking, registrasi, login, dan administrasi data).

Project ini dibuat menggunakan XAMPP (Apache + MySQL) tanpa framework, sehingga ringan dan mudah dijalankan di lokal maupun hosting.

🚀 Fitur Utama

🔐 Sistem Login & Logout
📝 Registrasi Peserta
👥 Manajemen Data Peserta
📊 Sistem Ranking / Penilaian
📤 Export Data ke Excel
🗂️ Upload File / Foto (folder uploads)
🧑‍⚖️ Halaman Admin & Juri
📍 Sistem Koneksi Database (db.php)
🖥️ Interface sederhana dan ringan

🛠️ Teknologi yang Digunakan

- PHP Native
- MySQL / MariaDB (XAMPP)
- HTML, CSS, JavaScript
- Bootstrap (jika digunakan di UI)
- PhpMyAdmin
